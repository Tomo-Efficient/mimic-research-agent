# EDA 到 PICO/PECO 提取规则

## 读取顺序

1. 先读数据来源、研究对象、纳排标准。
2. 再读变量字典、字段名、检测方法。
3. 再读描述统计、组间差异、相关性、回归或模型结果。
4. 最后读作者解释、局限性和可疑异常。

不要只根据显著性排序选题；先判断变量是否有临床含义、样本量是否足够、结局是否可解释。

## 必提字段

| 字段 | 提取内容 | 不足时处理 |
|---|---|---|
| 人群 | 来源、年龄、性别、地区、疾病场景、纳排标准 | 写 `not reported`，不要猜 |
| 疾病 | 标准诊断、亚型、分期、严重程度 | 名称不规范时保留原词并标注待确认 |
| 暴露变量 | 生物指标、治疗、行为、分组、检测结果、组学特征 | 区分连续变量和分类变量 |
| 结局变量 | 事件、诊断、疗效、预后、生存、复发、不良反应 | 明确方向和事件定义 |
| 分组变量 | 病例/对照、治疗组、风险组、亚型组 | 记录各组样本量 |
| 检测方法 | 实验平台、影像、问卷、量表、测序、算法 | 未报告时标注 |
| 显著变量 | p/FDR、效应量、CI、方向、模型调整变量 | 只写报告中有的统计量 |
| 样本量/事件数 | 总 N、各组 N、事件数、缺失数 | 事件数缺失时影响评分 |

## PICO/PECO 映射

- P：人群 + 疾病，不要只写疾病名。
- I/E：EDA 中最可解释、可验证、和结局相关的暴露变量。
- C：对照组、低暴露组、阴性组、常规治疗组或无该因素组。
- O：主要结局，优先选择事件定义清晰、临床意义明确的结局。

如果是观察性队列，优先使用 PECO；如果是干预或治疗比较，使用 PICO。

## 显著变量筛选

优先级：
1. 多变量模型仍显著且效应方向合理。
2. 有明确剂量反应或趋势。
3. 与主要结局直接相关。
4. 检测方法可靠。
5. 样本量和事件数可支撑分析。

降级信号：
- 只在单因素中显著。
- 多重检验未校正。
- 分组样本极不平衡。
- 事件数很少。
- 变量含义不清。
- 缺失值高。

## 早停规则

如果无法提取人群、疾病、暴露、结局中的任意两个核心要素，停止并要求补充 EDA 报告。

如果样本量或事件数完全缺失，可以继续生成检索式，但不能对 idea 的统计可行性给高分。
