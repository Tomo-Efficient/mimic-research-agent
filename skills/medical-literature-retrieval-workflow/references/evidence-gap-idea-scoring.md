# Evidence Gap Matrix 与 Idea 评分规则

## Gap 类型

| Gap 类型 | 判定标准 | 可转化性 |
|---|---|---|
| 人群 gap | 已有证据来自不同年龄、地区、疾病亚型或临床场景 | 高，若用户数据人群明确 |
| 暴露 gap | 研究过相近暴露，但未研究 EDA 中的具体变量/阈值/组合 | 高 |
| 结局 gap | 已有研究关注诊断或机制，但缺少预后、疗效或事件结局 | 中高 |
| 方法 gap | 已有主题研究，但检测方法、算法或统计策略不同 | 中 |
| 组合 gap | 单个变量有研究，但变量组合、交互或亚组证据少 | 高 |
| 验证 gap | 有发现但缺少外部验证或真实世界验证 | 高 |
| 机制 gap | 统计关联存在，但机制或中介路径不清 | 中 |
| 证据过密 | 已有大量直接研究和综述 | 低，除非数据有明显差异化 |

## Evidence Gap Matrix 字段

| 字段 | 说明 |
|---|---|
| EDA finding | 来自报告的变量组合或显著发现 |
| PICO/PECO | 对应的文献检索框架 |
| Recent original evidence | 近 3 年原始研究是否有直接证据 |
| Recent review evidence | 近 5 年综述是否覆盖 |
| Exact historical evidence | 不限时间精确组合是否已有直接证据 |
| Population match | 文献人群与用户数据人群是否匹配 |
| Exposure/method match | 暴露变量和检测方法是否匹配 |
| Outcome match | 结局是否匹配 |
| Gap type | 使用上表分类 |
| Idea potential | 高/中/低，并说明理由 |

## Idea 生成规则

每个 idea 必须同时满足：
- 能被用户数据验证。
- 和至少一个 EDA 发现对应。
- 有明确 PICO/PECO。
- 文献中不是完全空白到无法建立背景。
- 不只是“重复别人已做过的分析”。

优先 idea：
- 文献有相近证据，但缺少用户人群或检测方法。
- EDA 显示强相关且事件数足够。
- 临床结局明确。
- 可用常规统计模型验证。
- 可以形成清晰题名和摘要结构。

## 评分标准

每项 1-5 分。

| 维度 | 5 分 | 3 分 | 1 分 |
|---|---|---|---|
| 创新性 | 直接组合证据少，有清晰差异化 | 有一定重复，但角度可调整 | 已被大量研究覆盖 |
| 数据可验证性 | 变量、分组、结局、样本量齐全 | 关键变量可用但需清洗或补充 | 关键变量缺失 |
| 临床意义 | 影响诊断/预后/治疗决策 | 有临床解释但应用弱 | 临床意义不清 |
| 统计可行性 | 事件数和模型复杂度匹配 | 可做基础模型但有限制 | 事件数不足或偏倚严重 |
| 发表潜力 | 题目清晰、gap 明确、结果可讲述 | 适合短文或探索性研究 | 发表叙事弱 |
| 风险 | 低偏倚、低过拟合、低伦理/解释风险 | 有可控限制 | 高混杂、高缺失或不可解释 |

## 风险提示

必须降低风险评分的情况：
- 事件数 < 10/变量数。
- 主要暴露和结局来自同一时间点，难以解释方向。
- 缺少关键混杂因素。
- 多重检验后显著性可能消失。
- 检测方法不稳定或不可复现。
- 文献已有高度一致直接证据，创新不足。
